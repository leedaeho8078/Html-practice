3. link태그<br>
   -> 링크를 만들기 위한 태그 a태그+속성 href="경로"

4. img태그<br>
   -> img 불러오는 태그, 종료 태그가 없다, alt 속성->사진이 안보이는 경우 대체, 잘 작성하지 않음
   -> css를 구성하는 방법 중 1) style을 태그안에 -> inline 속성

5. br태그<br>
   -> 닫는 태그가 없다. 줄바꿈 용
   -> 닫는 태그가 없다. -> empty태그

6. style속성 <br>
   -> color,font-size -> attribute(속성값)

7. backgroud-color<br>

8. font-family<br>

9. font-size<br>

10. text-align<br>

11. i,em<br>
